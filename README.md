# 📋 ЧТО ТЕБЕ ПОНАДОБИТСЯ

Перед началом убедись что у тебя есть:
- Компьютер (Windows, Mac или Linux)
- Интернет
- Аккаунт в Telegram
- 15 минут времени

---

# 🔑 ШАГ 1 — Получи токен Telegram бота (5 минут)

**1.1.** Открой Telegram и в поиске найди бота **@BotFather**

**1.2.** Нажми **START** или напиши `/start`

**1.3.** Напиши команду:
```
/newbot
```

**1.4.** BotFather спросит **имя бота** — это то, что будут видеть люди.
Например: `Мой ИИ Ассистент`

**1.5.** Потом спросит **username** — это уникальное имя, заканчивается на `bot`.
Например: `moy_ai_bot` или `superai_helper_bot`

> ⚠️ Username должен быть уникальным. Если занят — придумай другой.

**1.6.** BotFather пришлёт сообщение с токеном вида:
```
1234567890:AAFfGhIjKlMnOpQrStUvWxYz1234567890
```

**Скопируй этот токен — он тебе нужен!**

---

# 🔑 ШАГ 2 — Получи бесплатный ключ Groq (3 минуты)

Groq — это бесплатный сервис с мощным ИИ (Llama 3.3). Регистрация за 2 минуты.

**2.1.** Открой браузер и зайди на: **https://console.groq.com**

**2.2.** Нажми **Sign Up** (Зарегистрироваться)

**2.3.** Зарегистрируйся через Google аккаунт или email

**2.4.** После входа в левом меню найди **API Keys**

**2.5.** Нажми **Create API Key**

**2.6.** Введи любое название, например `telegram-bot`

**2.7.** Скопируй ключ вида:
```
gsk_abcdefghijklmnopqrstuvwxyz1234567890
```

> ⚠️ Ключ показывается только один раз! Сразу сохрани его.

# 🔑 ШАГ 3 — Узнай свой Telegram ID

Это нужно чтобы стать администратором бота.

**3.1.** В Telegram найди бота **@userinfobot**

**3.2.** Напиши ему `/start`

**3.3.** Он пришлёт твой ID — число вида `123456789`

**Запомни это число.**

# 💻 ШАГ 4 — Установи Python

### Если у тебя Windows:

**4.1.** Зайди на **https://python.org/downloads**

**4.2.** Нажми большую кнопку **Download Python 3.12.x**

**4.3.** Запусти скачанный файл

**4.4.** ⚠️ ВАЖНО: Поставь галочку **"Add Python to PATH"** внизу окна установщика!

**4.5.** Нажми **Install Now** и дождись завершения

**4.6.** Проверь установку — открой командную строку (Win+R → напиши `cmd` → Enter) и введи:
```
python --version
```
Должно написать что-то вроде `Python 3.12.3`

### Если у тебя Mac:

**4.1.** Открой Terminal (через Spotlight: Cmd+Space → Terminal)

**4.2.** Введи:
```bash
brew install python3
```
Если brew не установлен, сначала:
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

**4.3.** Проверь:
```bash
python3 --version
```

### Если у тебя Linux (Ubuntu/Debian):

```bash
sudo apt update
sudo apt install python3 python3-pip python3-venv -y
python3 --version
```

# ⚙️ ШАГ 6 — Создай файл с настройками

**6.1.** В папке `tgbot_free` найди файл `.env.example`

**6.2.** Скопируй его и назови копию `.env` (без слова "example")
- Windows: скопируй файл, переименуй
- Mac/Linux: `cp .env.example .env`

**6.3.** Открой файл `.env` в любом текстовом редакторе
(Блокнот, VS Code, Notepad++ — что угодно)

**6.4.** Заполни три строки своими данными:

```
BOT_TOKEN=сюда_вставь_токен_от_botfather
GROQ_API_KEY=сюда_вставь_ключ_от_groq
ADMIN_IDS=сюда_вставь_свой_telegram_id
```

**Пример как должно выглядеть:**
```
BOT_TOKEN=1234567890:AAFfGhIjKlMnOpQrStUvWxYz1234567890
GROQ_API_KEY=gsk_abcdefghijklmnopqrstuvwxyz1234567890
ADMIN_IDS=987654321
```

**6.5.** Сохрани файл.

# 📦 ШАГ 7 — Установи зависимости (библиотеки)

Открой терминал/командную строку **в папке бота**:

**На Windows:**
1. Открой папку бота в Проводнике
2. В адресной строке напечатай `cmd` и нажми Enter
3. Введи команды по очереди:

```cmd
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

**На Mac/Linux:**
```bash
cd /путь/до/tgbot_free
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

> Установка займёт 1-2 минуты. Ты увидишь много строк — это нормально.

**Как понять что всё установилось:** последние строки будут вроде:
```
Successfully installed aiogram-3.13.1 groq-0.11.0 aiohttp-3.10.10 ...
```

# ⚠️ ЧАСТЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### "python не является внутренней командой" (Windows)
→ Python не добавлен в PATH. Переустанови Python, **поставив галочку "Add to PATH"**.

### "ModuleNotFoundError: No module named 'aiogram'"
→ Виртуальное окружение не активировано. Выполни:
```cmd
venv\Scripts\activate    # Windows
source venv/bin/activate  # Mac/Linux
```

### "❌ BOT_TOKEN не задан в .env файле!"
→ Проверь что файл называется `.env` (не `.env.txt` или `.env.example`).
→ Убедись что токен вставлен без пробелов.

### "❌ GROQ_API_KEY не задан в .env файле!"
→ Проверь что ключ начинается с `gsk_`
→ Убедись что нет лишних пробелов до и после ключа.

### Бот не отвечает в Telegram
→ Убедись что в консоли нет ошибок.
→ Попробуй написать `/start` снова.
→ Проверь что не заблокировал бота.

### "Conflict: terminated by other getUpdates request"
→ Бот уже запущен в другом месте. Закрой другой терминал или перезапусти.
